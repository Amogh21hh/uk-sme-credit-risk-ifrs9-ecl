## Version A — Credit Risk Analyst roles (preferred)

**UK SME Credit Risk Scorecard & IFRS 9 ECL Model** | R, Excel, Python | [GitHub link]

- Built an end-to-end IFRS 9 expected credit loss (ECL) model on a 10,000-loan UK SME portfolio, producing PD, LGD, EAD and stage allocations aligned with PRA SS3/17 and IFRS 9 paragraph 5.5
- Developed a logistic regression PD scorecard using WoE binning and Information Value screening; achieved **AUC 0.77, Gini 0.54 and KS 0.42** on a stratified hold-out, with Population Stability Index monitoring for deployment readiness
- Implemented IFRS 9 staging logic (Stage 1 / 2 / 3) with SICR triggers (PD threshold + DPD), lifetime PD conversion, and a 3-scenario probability-weighted macro overlay producing a **GBP 204m ECL on GBP 973m EAD (20.9% coverage)**
- Delivered a 5-tab Excel ECL model with 7,000+ live formulas, conditional staging and editable scenario weights, plus an interactive HTML dashboard with sector and stage views
